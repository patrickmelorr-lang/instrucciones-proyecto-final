Objetivo general

Crear notebooks individuales que demuestren relaciones lineales (y = ax + b) usando APIs en tiempo real, en diferentes ámbitos: software, finanzas, música, criptomonedas y GitHub.

  • Celda 2 imports + solicitud API key (segura)
  • Celda 3 ≥3 funciones con docstring y comentarios
  • Celda 4 ejecución, regresión y estadísticas
  • Celda 5 gráficas (PNG+SVG), JSON, dashboard
  • Archivos generados: PNG, SVG, JSON
